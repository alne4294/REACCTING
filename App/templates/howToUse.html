How to Run This App

 

The sourcecode for the app can be found here.  Since it is private, you will need to be requested to be added as a collaborator.

Issues with this app can be addressed to Alexia Newgord at alexia@alexianewgord.com.

 

Steps to run (from the server side)

 

  1. Install Mongo Database if you don't have it.  See this installation directions here.  This app was developed on version 2.6.7.
  2. Import the PEMs data dump by running: 
    mongorestore --collection p1 --db pems PEMs_data/dump/pems/p1.bson
  3. Start the MongoDB daemon with: 
    mongod
  4. Make sure your machine has all of the requirements by running the following from the base folder of the app.  
    pip install -r requirements.txt
  5. Run the application with: 
    python app.py
  6. If there are errors, you may be missing some dependencies... go ahead and install those as well.
  7. This app was developed using Python 2.7.6 and has only been tested with Firefox, Safari, and Chrome.
  8. The password can be changed in the app.py file.
